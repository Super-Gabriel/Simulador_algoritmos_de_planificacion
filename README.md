## Simulador de algoritmos de planificación de procesos

Este proyecto es un simulador de algoritmos de planificación de procesos desarrollado en Python. Implementa tres algoritmos clásicos de planificación: FIFO (First-In, First-Out), Round Robin y SJF (Shortest Job First). El simulador permite visualizar la ejecución de los procesos, calcular métricas de rendimiento (tiempo de espera, tiempo de respuesta y tiempo de retorno) y comparar el comportamiento de los algoritmos bajo diferentes condiciones.

## Uso

para ejecutar el simulador necesitamos clonar el repositorio, situarnos en la carpeta del proyecto y ejecutar el archivo `main.py`, es decir, ejecutar los siguientes comandos:

- git clone `https://github.com/Super-Gabriel/Simulador_algoritmos_de_planificacion.git`
- cd `Simulador_algoritmos_de_planificacion`
- python `main.py`
